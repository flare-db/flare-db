Flare implements the Apache Beam portability APIs and executes Beam
pipelines using a lightweight Rust runtime.

Apache Beam provides a powerful unified model for batch and stream
processing. The goal of Flare is to make running Beam pipelines and
data stream processing cost-efficient and operationally simple.

## Status

🌱 Early stage and work in progress.

The initial version will focus on correctly running Apache Beam
pipelines on a single node. Follow along and star the repo 🌟.
